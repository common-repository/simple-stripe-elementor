Simple Stripe Elementor, agrega un botón de Stripe a Elementor, de forma simple y sin tocar código.

== Description ==
Simple Stripe Elementor, simplifica el proceso de agregar botones para pagos desde Stripe. Agregando un toque de personalización realmente sencillo.
Creas el producto en Stripe, obtienes su ID y es TODO. Simple Stripe Elementor, hace el resto, agregando un nuevo Widget a tu set de Elementor.

== Installation ==
1. Descargar e instalar Simple Stripe Elementor
1. Activar el Plugin.
1. Ir al editor de Elementor y configurar el Widget.
